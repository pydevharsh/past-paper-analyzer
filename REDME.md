# 📊 Past Paper Analyzer & Smart Study Planner

## 🚀 Overview

Past Paper Analyzer & Smart Study Planner is a simple AI-inspired system that analyzes past exam papers to identify high-frequency topics and generate a prioritized study plan.
It helps students focus on important concepts and prepare more effectively for exams.

---

## 🎯 Problem Statement

Students often struggle to identify important topics from multiple past question papers.
Manual analysis is time-consuming and inefficient, leading to poor prioritization and missed high-yield topics.

---

## 💡 Solution

This project provides a lightweight solution that:

- Analyzes past paper text
- Detects topic frequency
- Identifies high-priority topics
- Generates a smart study plan

---

## ⚙️ Features

- Multi-paper text analysis
- Topic frequency detection
- High-yield topic identification
- Topic ranking based on importance
- Smart study plan generation
- Simple and fast CLI-based interface
- Clean and readable output
- Scalable for future AI/NLP enhancements

---

## 🧠 How It Works

1. User inputs past paper questions (text format)
2. System processes and cleans the data
3. Topics are detected using keyword matching
4. Frequency of each topic is calculated
5. Important topics are ranked
6. A study plan is generated based on priority

---

## 📥 Input

- Past question papers (text format)

## 📤 Output

- Topic frequency analysis
- Important topics list
- Smart study plan
